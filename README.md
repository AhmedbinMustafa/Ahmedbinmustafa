### Research and Project Highlights

* **Wood Dust Particle Prediction (Python)**
Modeled terminal velocities and particle distance using the Schiller-Naumann equation, Reynolds number, and Drag Coefficients in Python to analyze wood dust behavior.

* **Chemical Recycling of Plastics**
Conducted PET hydrolysis and PMMA (polymethyl methacrylate) depolymerization to recover monomers. Validated the quality of recycled materials using IR spectroscopy and reference datasets

* **Monitoring Parameters in Municipal Wastewater Treatment**
Quantified nitrogen species ($NO_{2}^{-}$, $NO_{3}^{-}$, TNb) using UV/VIS spectrometers (Specord 50) and a TNb analyzer. Established linear calibration functions according to the Lambert-Beer law.
